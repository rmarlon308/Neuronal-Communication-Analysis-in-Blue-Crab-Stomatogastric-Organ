Práctica Teoría de la Información

El código para la resolución para esta práctica se conforma de 7 archivos.
- Los siguientes 3 archivos hacen referencia a la codificación temporal de spikes:
	- analisis_control.ipynb: Este archivo contiene la codificación del archivo correspontiente a la etapa de control.
	- analisis_gaba.ipynb: Este archivo contiene la codificación del archivo correspontiente a la etapa de GABA.
	- analisis_recuperacion.ipynb: Este archivo contiene la codificación del archivo correspontiente a la etapa de recuperación.
- Los siguientes 3 archivos hacen referencia a la codigicación SAX de las señales:
	- analisis_control_sax.ipynb: Este archivo contiene la codificación SAX del archivo correspontiente a la etapa de control.
	- analisis_gaba_sax.ipynb: Este archivo contiene la codificación SAX del archivo correspontiente a la etapa de GABA.
	- analisis_recuperacion_sax.ipynb: Este archivo contiene la codificación SAX del archivo correspontiente a la etapa de recuperación.
- El archivo funcs.py contiene dos funciones las cuales son utilizadas para realizar el cáculo de los umbrales y la obteción de gráficas de la codificación.
